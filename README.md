# AI Startup Simulator 🚀

This project is an AI-based Startup Simulator built using Python.

## Features
- Simulates real-world startup decisions
- Actions: hire, develop, marketing, save, loan
- AI agent automatically makes decisions
- Reward system based on business growth
- Startup health and status tracking

## Tech Used
- Python
- HTTP Server
- JSON API
- Reinforcement Learning Logic

## How to Run
1. Install Python
2. Run: python server.py
3. Open:
   http://127.0.0.1:8000/start
   http://127.0.0.1:8000/ai

## Project Goal
To simulate startup decision-making using AI and reward-based learning.
